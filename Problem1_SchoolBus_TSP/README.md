# 🚍 School Bus Route Optimization (TSP)

## 📌 Problem Description
This project solves the Travelling Salesman Problem (TSP) to optimize a school bus route.  
The goal is to visit all stops exactly once and return to the starting point (school) while minimizing total distance.

---

## 🧠 Algorithm Used
- Nearest Neighbor (Greedy Algorithm)

---

## 📂 Files
- main.py → Runs the program
- tsp_solver.py → TSP algorithm
- utils.py → Output formatting
- data.py → Sample input
- visualization.py → Route plotting
