# 🧠 Smart Job Scheduling using Minimax Algorithm

## 📌 Problem Description
This project uses the Minimax algorithm to determine the optimal order of scheduling jobs under uncertain delays.

The scheduler (MAX player) tries to minimize total completion time, while the environment (MIN player) introduces worst-case delays.

---

## 🧠 Algorithm Used
- Minimax Algorithm (Decision Making)

---

## 📂 Files
- main.py → Runs the program
- minimax.py → Core Minimax logic
- scheduler.py → Scheduling controller
- utils.py → Output formatting
- data.py → Sample job data
